## How it works

J K Flip-Flop

## How to test

The flip-flop's clock is IN1,
J and K are IN0 and IN2, respectfully.

## External hardware

Outputs are OUT0 and OUT1

## Pinout

| Pin  | Description                             |
| ---- | --------------------------------------- |
| IN0  | J input                                 |
| IN1  | Clock input (rising edge)               |
| IN2  | K input                                 |
| ---- | --------------------------------------- |
| OUT0 | Q output                                |
| OUT1 | \Q Inverted output (complement of Q)    |
